# Privacy vs Evidence: Behavioral and Digital Forensic Analysis of the Sanchar Saathi Mobile Application

# 🔐 Sanchar Saathi Mobile Security & Forensic Analysis

> 📌 End-to-end analysis of an Android application covering reverse engineering, runtime behavior, network security, and forensic investigation.

---

## 📖 About This Project

This repository contains a complete technical analysis of the **Sanchar Saathi mobile application**, performed to understand its internal architecture, security mechanisms, and data handling behavior.

The project combines multiple domains of cybersecurity including:

* 🔍 Static Analysis (MobSF)
* 🧬 Reverse Engineering (APKTool, JADX)
* ⚙️ Runtime Analysis (ADB, Logcat, Dumpsys)
* 🌐 Network Analysis (Burp Suite, SSL Pinning)
* 🧪 Forensic Analysis (Magnet AXIOM, MOBILedit)

---

## 🎯 Purpose

The main objective of this project is to:

* Analyze application behavior from a security perspective
* Identify how data is collected, processed, and transmitted
* Evaluate implemented protection mechanisms
* Understand limitations of forensic data extraction
* Study the balance between user privacy and investigation capabilities

---

## 🧠 Key Highlights

* Hybrid architecture (Flutter + Native Android)
* Secure communication using HTTPS and SSL pinning
* Background data synchronization mechanisms
* Controlled data collection and storage
* Limited forensic visibility in non-rooted environments

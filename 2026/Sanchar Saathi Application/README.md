# Privacy vs Evidence: Behavioral and Digital Forensic Analysis of the Sanchar Saathi Mobile Application

## 📌 Project Overview
This project investigates the forensic visibility and privacy implications of the Sanchar Saathi mobile application developed by the Government of India. The study focuses on understanding how privacy-preserving application design impacts the availability of digital forensic evidence at the user-device level.

The analysis was conducted under controlled, non-rooted conditions to simulate real-world forensic constraints.

---

## 🎯 Objectives
- Analyze forensic artifact availability in a privacy-focused mobile application
- Evaluate application behavior under different runtime scenarios
- Study the impact of sandboxing, encryption, and SSL pinning
- Identify limitations of user-side forensic investigations
- Compare logical vs root-level forensic possibilities
This project highlights the growing challenge of balancing **user privacy vs forensic evidence availability**.
